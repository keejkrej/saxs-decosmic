# Installation
```
cd xrd-decosmic
pip install -e .
```
# Usage
```
python -m xrd_decosmic.main
```
- File - Load Data - 
Select first image in the series
- File - Load Parameters (Optional)
- Start
- Select plot type
- Plot
- File - Save Data - 
Select folder to save
- File - Save Parameters (Optional)
