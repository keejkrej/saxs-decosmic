# XRD Decosmic

A tool for removing high energy background from XRD 2D images.

## Installation

```bash
git clone https://github.com/keejkrej/xrd-decosmic.git
cd xrd-decosmic
pip install -e .
```

## Usage

```bash
python -m xrd_decosmic.cli \
--input-file /path/to/first_filename.tif \
--output /path/to/output/
```
